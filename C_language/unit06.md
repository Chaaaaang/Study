# 디버거 사용하기

* 디버거(debugger) : 버그(bug)를 제거하는(de-) 도구라는 뜻

  -> 프로그램 내부 상황을 손쉽게 파악할 수 있도록 만듦.

  * 중단점 : 브레이크 포인트로 소스 코드의 특정 지점에서 프로그램의 실행을 멈추는데 사용함.
  * 디버깅을 하면 한 줄씩 가능한데 이를 통해 변수에 들어간 값 등 확인 가능.

* (참고) 디버깅 주요 키
  * 중단점 삽입/삭제: F9
  * 디버깅 시작: F5
  * 디버깅 중지: Shift+F5
  * 프로시저 단위 실행: F10
  * 한 단계씩 코드 실행: F11
